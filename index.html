sql-window-function-desc

发表于 2019-10-10

SQL数据结构

术语

  • window function 窗口函数
  • window function type 窗口函数类型/聚类函数 类似max,mean
  • window specification 窗口规范，关于窗口的定义
  • window partition 窗口分区，根据列进行分区
  • window order 窗口排序，针对结果集进行排序的
  • window frame 滑动窗口，具备上限跟下限
  • window function = window function type over window specification
  • window specification = window partition | window order | window frame
